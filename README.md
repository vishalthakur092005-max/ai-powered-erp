# Amdox ERP

Enterprise Resource Planning system built with MERN stack.

## Project Structure

```
amdox-erp/
├── frontend/          # React + TypeScript frontend
├── backend/           # Node.js + Express backend
└── docs/              # Documentation
```

## Quick Start

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run dev
```

## Tech Stack

- **Frontend**: React 18, TypeScript, Vite, TailwindCSS
- **Backend**: Node.js, Express, TypeScript
- **Database**: MongoDB
- **State Management**: Redux Toolkit

## Modules

- Finance (GL, AP, AR, Payments)
- HR (Employees, Leave, Attendance, Payroll)
- Supply Chain (Purchase Orders, Inventory, Vendors)
- Projects (Tasks, Resources, Budgets)
- Dashboard & Reports
- Settings & User Management
